・2017/09/03 author : K.H
　9/1に実施したGitHub解説会でプロキシの設定を行いましたが、当日付け焼刃の作業だったため、ご自宅で作業されている場合でもSync（リポジトリ同期）に際してなんがしかのエラーが出ているかもしれません。その場合、下記手順に従って、プロキシ設定のリセットを行ってください。

  -GitShellを使う場合
　　・エディタで、　git config --global --unset http.proxy　と打込、Enter

　-Windows PowerShellを使う場合
　　・エディタで、　PS> netsh winhttp reset proxy　と打込、Enter


　プロキシのリセットが終わったら、
GitHubアプリの 「Changes」 ビュー上、「〜 Changes」（〜は変更ファイル数）を右クリック
 > 「discard all changes」でご自身の全ての変更を破棄
 > 「Sync」ボタンをクリックして同期してください。
